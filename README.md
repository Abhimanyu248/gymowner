# Gym Member Manager

A simple React Native app for managing gym members.

## Features

- Store member joining date
- Store membership expiry date
- Display member image
- Select membership type
- View all member records in a mobile-friendly list

## Run the app

1. Install dependencies:

```bash
npm install
```

2. Start Expo:

```bash
npm start
```

3. Open it on Android, iOS, or web from the Expo menu.

## Notes

- The current version stores data in local component state.
- Member images are added using an image URL field.
- You can extend this next with local storage, camera upload, search, edit, and delete.
